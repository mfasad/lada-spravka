Режим работы и правила посещения

Столовая работает в непрерывном режиме, но с чётким расписанием по типам питания. Это связано с графиком смен на производстве:

Вид питания Время работы Особенности
Завтрак 07:30–09:30 Горячие блюда до 09:00, после — только буфет
Обед 11:30–14:00 Пиковое время: 12:00–13:00 (очереди до 15 минут)
Полдник 15:00–16:30 Выпечка, десерты, чай/кофе
Ужин 17:00–19:00 Упрощённое меню (без супов)
Ночная смена 23:00–01:00 Только для сотрудников с пропуском «Н»

Ключевые правила, которые часто нарушают посетители:

  1. Запрет на вынос еды: нельзя брать блюда в контейнерах (исключение — корпоративные мероприятия с разрешения администрации).
  2. Одежда: обязательна сменная обувь или бахилы для сотрудников цехов.
  3. Оплата: терминалы принимают только карты МИР — Visa/MasterCard не работают.
  4. Бронь столов: для групп от 10 человек требуется предварительная заявка (за 2 дня).

Для гостей города действуют дополнительные ограничения:

  • 🕒 Время посещения — только в обеденные часы (11:30–14:00).
  • 🍽️ Запрещено пользоваться посудой для самообслуживания (обслуживание только через раздаточную линию).
  • 📸 Фотосъёмка разрешена только без сотрудников в кадре.
💡

Самое удобное время для посещения без очередей — 11:30–12:00 или 13:30–14:00. В пиковые часы (12:00–13:00) среднее время ожидания — 10–15 минут.

FAQ: ответы на частые вопросы о столовой «Лада» в Балаково

Можно ли посетить столовую без пропуска на территорию завода?

Да, но для этого нужно:

  1. Предварительно оформить разовый пропуск через сайт АвтоВАЗа (раздел «Посетители»).
  2. При себе иметь паспорт.
  3. Посещать столовую только в обеденное время (11:30–14:00).

С 2026 года для гостей действует ограничение: не более 2 посещений в месяц.

Какие документы нужны для оформления корпоративной карты на питание?

Сотрудникам АвтоВАЗа достаточно:

  • Заявление на имя начальника отдела кадров.
  • Копию трудового договора.
  • Фото 3×4 (для изготовления карты).

Карта изготавливается 3 рабочих дня. Баланс пополняется автоматически из зарплаты (можно выбрать сумму от 1000 до 5000 рублей в месяц).

Можно ли заказать еду из столовой навынос для мероприятия?

Да, но только для корпоративных мероприятий АвтоВАЗа. Правила:

  • Минимальный заказ — 10 порций.
  • Оплата только по безналу (корпоративная карта или перевод на счёт столовой).
  • Заявка подаётся за 3 дня через email balakovo-catering@lada.ru.

Для частных лиц услуга не предоставляется.

Есть ли в столовой веганское меню?

С 2023 года введено вегетарианское меню, но полноценного веганского нет. Из доступных опций:

  • Салаты без мяса/рыбы/яиц (например, «Цезарь» с грибами).
  • Гарниры: рис, гречка, овощи на гриле.
  • Супы: щи зелёные, борщ постный (по средам).

По запросу можно приготовить блюда без молочных продуктов (нужно предупредить за день).

Как вернуть деньги, если не понравилось блюдо?

Для возврата средств:

  1. Сохраните чек (электронный или бумажный).
